# FlickDB

A single-page web app using [AMP](https://developers.google.com/amp) to search for movies using the [OMDb](http://www.omdbapi.com/) API.

## Usage

Create an `api-key.json` file:
```json
{
  "omdbKey": "<your key here...>"
}
```

Then launch the web-app
```
yarn install
yarn run dev:run
```
